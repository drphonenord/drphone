# Logos des marques — Temporaires (à remplacer)
Ces SVG sont **des placeholders** pour que la section “Marques” s’affiche immédiatement.
Pour mettre les **logos officiels**, remplace simplement les fichiers ci-dessous par les SVG/PNG officiels (mêmes noms de fichier), puis redéploie sur Netlify.

Fichiers à remplacer :
- assets/brands/apple.svg
- assets/brands/samsung.svg
- assets/brands/huawei.svg
- assets/brands/xiaomi.svg
- assets/brands/oppo.svg
- assets/brands/oneplus.svg
- assets/brands/google.svg
- assets/brands/sony.svg
- assets/brands/nokia.svg
- assets/brands/motorola.svg
- assets/brands/realme.svg
- assets/brands/honor.svg
- assets/brands/asus.svg

Astuce : tu peux aussi déposer des **PNG** (ex. `apple.png`). Si le code HTML pointe vers `.svg`, garde les **mêmes noms** et extensions pour éviter de toucher au code.
