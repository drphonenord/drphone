# Logos officiels (optionnel, pour hébergement local)
Cette page utilise déjà les LOGOS OFFICIELS via des liens Wikimedia (affichage immédiat).
Si tu préfères héberger en local sur Netlify :

1) Ouvre un terminal dans le dossier du site.
2) Exécute l’un des scripts ci-dessous pour télécharger les SVG dans `assets/brands/` :
   - macOS/Linux : `bash fetch-logos.sh`
   - Windows PowerShell : `.etch-logos.ps1`
3) Si tu veux repasser sur les fichiers locaux, remets les `src="assets/brands/*.svg"`
   (ou dis‑moi, je te fais la variante directement).

Remarques:
- Le logo Google: choisis soit l’icône "G", soit le mot‑symbole "Google".
  Actuellement le script prend l’icône "G".
- Logos = marques déposées; usage nominatif pour informer les clients.
